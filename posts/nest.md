## Iniciando Projeto Nest
Instalar NestJs
```bash
$ npm i -g @nestjs/cli
$ nest new project-name
```
Criar recurso de CRUD
```bash
nest g res resorce-name
```
Iniciar
```bash
npm run start:dev
```
